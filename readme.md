## Setting Up Default Project with Webpack, Babel and D3

1. `git clone https://github.com/lfarrell/basic-template.git new-name`
2. `cd new-name && npm install`

#### Running Dev Server

`npm start`

#### Production Build

`npm run-script prod` or `webpack -p -d`
